How This Website Works

HTML is the structure. It creates the sections, titles, buttons, and text.

CSS is the decoration. It controls colors, spacing, fonts, cards, and mobile layout.

Links are made with the <a> tag. The Sarah Brightman button is a link that opens YouTube in a new tab.

Cards are boxes created with CSS. They make the website feel elegant and organized.

Mobile-friendly design comes from this line:

<meta name="viewport" content="width=device-width, initial-scale=1.0" />

This tells the browser: “Please adapt this page to the size of the phone.”
